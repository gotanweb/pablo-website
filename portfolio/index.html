Montfrut Website
Project Overview

I designed and developed Montfrut's official website, a fully responsive platform created to strengthen the brand's digital presence and showcase handcrafted chocolate products with a professional, modern UI. The site integrates dynamic product galleries, category filtering, an interactive FAQ, testimonials, and a contact form connected to email handling.

Key Features
  • Dynamic product portfolio with category filtering using Isotope.
  • High-quality image galleries powered by GLightbox.
  • Interactive UI animations using AOS and Swiper.
  • Responsive design fully optimized for mobile, tablet, and desktop.
  • SEO-ready structure with meta tags and schema markup.
  • Contact form integration using PHP Email Form.
  • FAQ section with collapsible items built with Bootstrap.
  • Testimonials slider implemented with Swiper.
  • Google Maps, WhatsApp, and social media integration.
Technologies
HTMLCSSBootstrap
Outcomes

Delivered a polished and scalable online presence that increased customer engagement, improved communication through integrated contact channels, and provided a professional product showcase that enhanced brand credibility.
