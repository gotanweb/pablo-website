Montfrut Website
Project Overview

I designed and developed Montfrut’s official website, a fully responsive platform created to strengthen the brand’s digital presence and showcase handcrafted chocolate products with a professional, modern UI. The site integrates dynamic product galleries, category filtering, an interactive FAQ, testimonials, and a contact form connected to email handling.

The project focused on delivering a clean visual identity, fast performance, and a seamless mobile-first experience to support customer engagement and local sales.

Key Features
  • Dynamic product portfolio with category filtering using Isotope for an intuitive browsing experience.
  • High-quality image galleries powered by GLightbox, enabling previews for each product.
  • Interactive UI animations using AOS and Swiper for smooth transitions and carousels.
  • Responsive design fully optimized for mobile, tablet, and desktop.
  • SEO-ready structure with meta tags, schema markup and clean semantic HTML.
  • Contact form integration using PHP Email Form for message handling.
  • FAQ section with collapsible items built with Bootstrap components.
  • Testimonials slider implemented with Swiper for brand credibility.
  • Local business information including Google Maps link, WhatsApp contact, and social media integration.
Technologies Used

HTML CSS Bootstrap

Outcomes

The project delivered a polished and scalable online presence that increased customer engagement, improved communication through integrated contact channels, and provided a structured, professional product showcase that enhanced brand credibility and reduced dependence on social media as the primary sales channel, ultimately giving the business a more reliable and independent digital ecosystem.

E-Finance Project
Project Overview

Casa Dulman e-Finance is a web system developed to digitize and streamline the company’s purchase management process, replacing manual spreadsheets with structured data entry, automated validation, and centralized financial processing. The platform allows users to register invoices, debit notes, and credit notes, classify each purchase under predefined accounting categories, and generate monthly and annual balance reports, supporting remote access and reducing dependency on outdated on-premise tools.

Key Features
  • Management of accounting categories, suppliers, and purchase documents..
  • Automated calculation of monthly and annual purchase balances.
  • Data validation and integrity rules to prevent inconsistent records.
  • Dynamic MVC workflows using Laravel migrations, views, and controllers.
  • Support for remote access and future scalability toward payroll and billing modules.
Technologies Used

PHP Laravel Livewire MySQL Bootstrap

Outcomes

The system reduced manual processing time by organizing all purchase data within one structured interface, improved accuracy in monthly and annual balance creation, and established a scalable foundation for future financial modules, enabling the business to transition from Excel-based workflows to a more reliable and consistent digital process.
